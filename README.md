# ML PROJECT OVERVIEW:

## Installation:
* First Make a Python virtual envoirnment.

  ```
  python -m venv .venv
  ```

* Then activate it with
  ```
  .venv\Scripts\activate
  ```

* When it activates, your terminal prompt will change into:
  ```
  (env) C:\Users\Maham\my_project>
  ```

* Now, Install packages inside your environment
  ```
  pip install pandas numpy seaborn matplotlib scikit-learn catboost xgboost Flask 
  ```

* Save your dependencies (This will save your installed packeges into your project)
  ```
  pip freeze > requirements.txt
  ```

---
Now you are ready to go :)
